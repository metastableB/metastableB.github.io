ValDA: Valorant Data Analysis

ValDA is a simple python wrapper over the Valorant-MATCH-V1 API that can be used to interact with valorant DTO objects in python and can be used to interface to databases/files/jupyter etc.

This is purely a personal project to understand more about the game and to build visualizations to analyze game states.

This is NOT for public release and this page is a placeholder to satisfy Riot API Application Requirements (fingers crossed).
